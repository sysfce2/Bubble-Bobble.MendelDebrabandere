# Arche

## History
This project started as a course project for Programming 4 in DAE, where we learned how to make a small game engine.
I coded the game Bubble Bobble in that game engine as my final project for that course.
That code is still visible in the Bubble Bobble branch.


## Arche
Since then i wanted to keep working on the engine as a fun side project, so i renamed the engine to Arche and the game to Qualia.
I don't have a specific goal with this project yet, but my current idea is to have fun with game dev.
The idea is to have a game engine that can handle both 2D and 3D.
As a super long goal down the road i would maybe also experiment with building an editor for the engine. 

## Qualia
This is the game i want to build in the engine.
The concept for this game is based on the Danganronpa & Persona games, but with an original story.
The game itself will of course grow with the engine and vice versa.
